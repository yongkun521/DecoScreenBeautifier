# 商业化思路与定价备忘（2026-04-05）

> 用途：给我自己看的，不追求正式，只要结论明确。

## 一句话结论

- 这不是“大众软件”，而是“小而美的桌搭/副屏圈层软件”。
- 不建议走纯开源免费。
- 不建议走订阅制。
- 不建议把 Steam 当首发必须渠道。
- 最适合的路线：`免费社区版 + 一次性买断 Pro`，或者过渡期先直接卖一个低门槛买断版。

## 我对产品价值的判断

- 有价值点：
  - 专门做副屏，尤其是超宽、细长、异形比例。
  - 风格明确，不是普通监控面板，而是“赛博朋克副屏美化”。
  - 已经做到相对开箱即用，不像 Rainmeter / AIDA64 那样高度依赖折腾。
  - 有模板、样式预设、编辑器，具备“成品软件”潜力。
- 限制点：
  - 用户盘子不会特别大，属于桌搭/机箱屏/副屏玩家圈层。
  - 生态还不够强，模板库、分享生态、安装体验、用户手册仍有提升空间。
  - 现阶段卖点是“省事 + 好看 + 专门适配”，不是“功能最多”。

## 对标价格（只用于校准心理预期）

- Wallpaper Engine：`$4.99`
- FPS Monitor：`$12.99`
- DisplayFusion：`$39.99`
- AIDA64 Extreme：`€59.95 / 3 PCs`

我的理解：

- 我们现阶段不该按 AIDA64 的价格卖。
- 也不该压到“几块钱小玩具”。
- 目前最合理的位置，在 `Wallpaper Engine` 和 `FPS Monitor` 之间偏低一些。

## 定价结论

### 现在就拍板的价格

- **中国区标准价：`49 元`**
- **首发价 / 早鸟价：`39 元`**
- **海外标准价：`$7.99`**
- **海外首发价：`$5.99`**

### 我的定价原则

- 未来 6~12 个月，**不要高于 `59 元 / $8.99`**。
- 在模板生态、编辑器体验、安装稳定性没有明显跃迁前，**不要涨到 `79 元+`**。
- 也**不要低到 `19 元` 以下**，否则会把产品锚定成“廉价小工具”，不利于后续抬价。

## 推荐卖法

### 方案 A：我最推荐

- `Community`：免费
- `Pro`：`49 元`（首发 `39 元`）

Community 可以保留：

- 基础显示
- 少量模板
- 基础组件
- 基础编辑能力

Pro 负责卖：

- 完整模板库
- 更好的编辑器能力
- 更完整的副屏比例预设
- 更稳的打包版与分发体验
- 后续高级模板/主题包优先权益

### 方案 B：更省事的过渡方案

如果短期不想切免费/付费版本，先直接卖一个：

- **当前正式版统一卖 `49 元`**
- 首发 1~2 个月卖 `39 元`

这个方案简单，适合先验证有没有人愿意买。

## 不建议做的事

- 不做订阅制。
- 不做首发高价。
- 不把“开源”当成默认路线。
- 不把 Steam 当第一站。

原因很简单：

- 你的软件不是高频 SaaS，订阅会强烈劝退。
- 首发高价会放大用户对生态和稳定性的要求。
- 纯开源免费会过早放弃商业空间。
- Steam 更适合放大，而不是起步验证。

## 渠道顺序建议

### 第 1 阶段

- GitHub Releases / 官网下载页
- 国内自己的售卖渠道
- 用小范围用户先验证是否有人愿意付 `39~49 元`

### 第 2 阶段

- 补模板库、用户手册、安装指引、问题诊断
- 明确免费版和 Pro 的差异
- 开始积累案例图、桌搭展示图、用户反馈

### 第 3 阶段

- 再考虑 Steam
- 这时 Steam 是“放大器”，不是“救命渠道”

## 什么时候可以涨价

满足下面任意 3 条，再考虑从 `49` 涨到 `59`：

- 内置高质量模板明显增加（例如达到 15~20 套）
- 编辑器更顺手，真正适合普通用户
- 安装/启动/更新体验明显更稳
- 有用户案例和口碑沉淀
- 有模板分享或社区内容生态

## 我现在的最终拍板

- **短期就按 `39 元首发 / 49 元常规` 来看。**
- **海外就按 `$5.99 首发 / $7.99 常规`。**
- **短期不上 Steam 也完全没问题。**
- **等产品更成熟，再决定是否做 Community/Pro 分层和 Steam 上架。**
