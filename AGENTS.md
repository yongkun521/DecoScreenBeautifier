## Repository expectations

- 如果你不了解项目内容，可以阅读docs文件夹下的文档进行了解。
- 每次任务开始前，请阅读docs\Todolist.md来查看当前大致的功能完成进度。
- 每次任务完成后，请同步更新相关文档（主要是docs\Todolist.md，或者你创建的任务清单。）来记录进度。
- 你可以随时将你发现的问题记录到"docs\观察但未处理的问题.md"中。没有的话请创建。
- 如果我给你发送的请求比较复杂，你可以考虑在docs下创建新的.md文件，作为你的任务清单，以及备忘，避免你忘记。
- 我们有venv文件夹，如果你需要安装任何库，请安装到venv下。

## Repository Introduction
- 项目是一款面向电脑副屏的 CLI/TUI 赛博朋克美化软件，重点适配超宽、细长等非标准屏幕比例。系统以 Textual/Rich 为核心渲染框架，提供硬件/网络/时钟/音频/图像与 GIF 等模块化组件。它支持模板与字体/字符预设，便于一键套用不同布局、配色与风格，并通过可视化编辑器进行拖拽与参数调整。配置以 JSON5 持久化，包含全局缩放（字符像素密度）等参数，用于优化图像像素化与整体观感。
